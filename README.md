# 🛠️ Run and deploy your AI Studio app
This contains everything you need to run your app locally.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

# 🧠 Core Modules:
AI Forge Graphics
Enter your name and profession. Our AI synthesizes a professional layout and custom branding assets automatically.

Live Editing
Click any text on the card to edit it in real-time. Toggle between Standard, Centered, and Modern Split layouts.

Identity Vault
Manage multiple profiles. Each profile (Personal, Business, Family) has its own private repository for saved cards.

The 3-in-1 Package
Archive your design into the vault with one click. Use 'Archive All 3 Layouts' to save every variation simultaneously.

Export & Management
Download any archived card as a high-quality PNG or print-ready PDF. You can also delete specific designs or purge your entire vault to stay organized.

# 💡 Pro Tips
Connected Presence
Add a URL to generate a dynamic QR Code that connects your physical card to your digital presence.

Logo Assets
Use the 'Download Logo Assets' button to export your AI-generated brand mark for use in other documents.
